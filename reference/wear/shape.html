Watch face shape

Watch frames can be round or square, these functions allow to query which one is the case.


void setup() {
}      

void draw() {
  background(0);
  if (wearSquare()) {
    rect(0, 0, width, height);
  }  
  if (wearRound()) {
    ellipse(0, 0, width, height);
  }    
}
