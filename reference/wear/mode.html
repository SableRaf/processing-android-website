Watch face mode

A watch face can be either in ambient, always-on mode, or in interactive mode. Use these functions to determine the current mode:


void setup() {
}      

void draw() {
  if (wearInteractive()) {
    // draw full watch face
  }  
}


void setup() {
}      

void draw() {
  if (wearAmbient()) {
    // draw ambient mode watch face
  }  
}
